## рҹҡҖ Quick Start

```bash
# 1. Install
pip install tensorflow-macos tensorflow-metal   # macOS M-series
pip install -r requirements.txt

# 2. Train models
python train.py --dataset archive --model both

# 3. Run
python server.py --port 5001

# вҶ’ Open http://localhost:5001
```

**Docker:**
```bash
cp .env.example .env

# вҶ’ Open http://localhost:5000
```

## рҹҺ“ Classroom Mode вҖ” How It Works

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ    в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ    в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ    BEFORE    в”Ӯ вҶ’  в”Ӯ     DURING     в”Ӯ вҶ’  в”Ӯ    AFTER     в”Ӯ
в”Ӯ  Baseline    в”Ӯ    в”Ӯ  Live Tracking в”Ӯ    в”Ӯ  Assessment  в”Ӯ
в”Ӯ  capture     в”Ӯ    в”Ӯ  per student   в”Ӯ    в”Ӯ  + Report    в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ    в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ    в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

1. **Register students** вҖ” upload photo + name + roll number
2. **Start session** вҖ” enter teacher, subject, room
3. **Run phases** вҖ” click Before / During / After at the right time
4. **End session** вҖ” auto-generates full report with:
   - Engagement % per student per phase
   - Comprehension score (after vs before delta)
   - Trend: `improved` / `stable` / `declined`
   - Personalized recommendation per student
   - Class-level summary

## рҹӣ пёҸ API Reference

**WebSocket events:**
```
frame              вҶ’  Send base64 frame for analysis
detection_result   вҶҗ  Receive face boxes + emotions + student name
set_phase          вҶ’  Switch session phase (before/during/after)
```

**REST endpoints:**
```
GET  /api/status             Server health + model info
GET  /api/students           All registered students
POST /api/students           Register new student
POST /api/sessions           Start classroom session
POST /api/sessions/end       End session + generate report
GET  /api/sessions/:id/report   Fetch saved report
GET  /api/cameras            Detect available cameras
```

## вҡҷпёҸ Configuration (`.env`)

```env
PORT=5000
CONFIDENCE_THRESHOLD=0.45      # Lower = detect more faces
USE_ENSEMBLE=true              # Both models averaged
FACE_DETECTOR=dnn              # dnn (recommended) | haar
FACE_RECOGNITION_TOLERANCE=0.5 # Lower = stricter matching
PROCESS_FPS=10                 # Inference FPS (camera runs at 30)
```
